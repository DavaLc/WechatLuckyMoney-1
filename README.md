
# 微信红包

帮助你更快速方便的抢红包

&nbsp;

### 特性
 
 * 检测聊天列表、聊天页面 红包并自动打开

 * 悬浮开关，方便停用开启插件
 
 * 轻量安全、无需Root等权限，代码公开
 
&nbsp;
 
 ###  使用教程
 
 * 检查手机系统是否 4.3及以上版本
 
 * 打开APP，点击开启按钮
 
 * 根据提示，进入系统辅助功能页面，授权于本APP
 
 * 进入微信 聊天列表、或聊天详情页面
 
 * 无需其他操作，坐等红包
